# Objetivo del Video

Comprender cómo nace una aplicación ASP.NET Core utilizando .NET 9 antes de escribir la primera línea de lógica de negocio.

---

# Gran Idea

> Antes de escribir código, comprende cómo funciona la aplicación.

---

# Público Objetivo

- Desarrolladores Junior
- Estudiantes de programación
- Personas que comienzan con ASP.NET Core
- Desarrolladores provenientes de .NET Framework

---

# Prerrequisitos

- Visual Studio 2022
- .NET 9 SDK
- Conocimientos básicos de C#
- Ganas de aprender

---

# Resultado Esperado

Al finalizar este video el estudiante será capaz de:

- Comprender la estructura inicial del proyecto.
- Explicar Program.cs.
- Comprender el concepto de Builder.
- Diferenciar configuración de ejecución.
- Comprender el Request Pipeline.
- Identificar los archivos principales del proyecto.

---

# Objetivos Pedagógicos

Al finalizar el video el estudiante podrá responder:

1. ¿Qué sucede cuando presiono F5?

- No memorices Program.cs. Compréndelo.

- Una aplicación profesional comienza con una buena configuración.
