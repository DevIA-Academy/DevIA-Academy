# Objetivo

Definir las reglas visuales oficiales de DevIA Academy.

Toda pieza gráfica deberá respetar estas guías.

---

# Principios

La marca deberá transmitir:

- Ingeniería
- Profesionalismo
- Claridad
- Tecnología
- Confianza
- Aprendizaje

Nunca deberá transmitir:

- Caos
- Clickbait
- Saturación
- Infantilización
- Exceso de efectos

---

# Estilo Visual

Minimalista.

Moderno.

Inspirado en productos tecnológicos.

Interfaces limpias.

Mucho espacio en blanco.

Jerarquía clara.

# Logo

El logotipo deberá funcionar correctamente en:

- Fondo blanco

- Fondo oscuro

- Miniaturas

- GitHub

# Banner

Debe comunicar:

- Qué enseña DevIA Academy

- Valor diferencial

- Profesionalismo

---

# Miniaturas

Objetivos:

Máxima legibilidad.

Pocos elementos.

Mucho contraste.

Un solo mensaje.

No utilizar más de tres colores principales.

---

# Animaciones

Suaves.

Profesionales.

No invasivas.

Siempre al servicio del aprendizaje.

---

# Uso de IA

La IA podrá utilizarse para generar recursos.

Nunca sustituirá decisiones de diseño.

Todo resultado será revisado manualmente.
