# Definition of Done - Videos

Un episodio de DevIA Academy se considera terminado cuando:

## Contenido

□ El guion está aprobado.

□ Se respetó DAPS.

□ Existe una promesa clara.

□ El cierre responde la promesa.

---

## Técnica

□ Todo el código funciona.

□ No existen errores.

□ El repositorio fue actualizado.

---

## Producción

□ Grabación finalizada.

□ Edición finalizada.

□ Branding aplicado.

---

## Publicación

□ Video publicado.

□ Recursos publicados.

□ Shorts publicados.

□ TikTok publicado.

---

## Calidad

□ Pasó todos los Quality Gates.

□ Revisión final aprobada.
