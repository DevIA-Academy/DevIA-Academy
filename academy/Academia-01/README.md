# Academia 1 - Fundamentos de Programación

Esta academia está enfocada en enseñar los principios fundamentales de la programación utilizando C#.

## Contenido

* Variables
* Tipos de datos
* Operadores
* Condicionales
* Ciclos
* Métodos
* Arreglos
* Debugging

## Proyecto Final

Calculadora en Consola.

## Objetivo

Aprender la lógica de programación necesaria para construir aplicaciones desde cero.
