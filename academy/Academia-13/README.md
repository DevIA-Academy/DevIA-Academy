# Academia 13 - Proyecto Empresarial Full Stack

Esta academia integra todos los conocimientos adquiridos durante el recorrido de DevIA Academy.

## Tecnologías

* .NET
* ASP.NET Core
* Angular
* SQL Server
* Entity Framework Core
* Azure
* Docker
* GitHub
* Inteligencia Artificial

## Proyecto Final

Desarrollo de una plataforma empresarial completa utilizando arquitectura moderna y buenas prácticas.

## Objetivo

Construir una aplicación profesional que sirva como portafolio para entrevistas técnicas y demuestre las habilidades adquiridas durante toda la academia.
