# Academia 3 - Git y GitHub

Aprenderás a utilizar Git como sistema de control de versiones y GitHub como plataforma de colaboración.

## Contenido

* Instalación de Git
* Repositorios
* Commits
* Branches
* Merge
* Pull Requests
* GitHub Flow

## Proyecto Final

Repositorio colaborativo utilizando Git.

## Objetivo

Trabajar como un desarrollador profesional utilizando control de versiones.
