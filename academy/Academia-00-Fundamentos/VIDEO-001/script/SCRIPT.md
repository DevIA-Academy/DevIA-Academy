# Descripción

En este episodio construiremos desde cero una API profesional utilizando ASP.NET Core y .NET 9.

Este proyecto no será un ejemplo aislado. Será la base de una aplicación que evolucionará durante toda la Academia 00.

El objetivo es que el estudiante aprenda no solo a crear una API, sino también a comprender la estructura de un proyecto profesional desde el primer día.

---

# Público objetivo

- Personas sin experiencia en ASP.NET Core.
- Estudiantes de programación.
- Desarrolladores que vienen de otras tecnologías.
- Personas que desean aprender construyendo proyectos reales.

---

- .NET 9 SDK

# Escena 4

## Nombre

¿Qué es una Web API?

## Tipo

🔍 Explain

## Duración estimada

2 minutos

## Objetivo pedagógico

Explicar de forma sencilla qué es una Web API y cuál es su propósito dentro de una aplicación moderna.

## Idea clave

Una Web API permite que diferentes aplicaciones se comuniquen entre sí mediante peticiones HTTP.

## Error común

Pensar que una API es una base de datos o una aplicación con interfaz gráfica.

## Consejo profesional

Una API representa la capa de comunicación de un sistema. Debe enfocarse en exponer funcionalidades, no en almacenar información ni en mostrar interfaces de usuario.

## Visual

Diagrama animado.

Cliente

↓

HTTP Request

↓

Web API

↓

Lógica

↓

HTTP Response

↓

Cliente

Aparecen ejemplos de:

• Aplicación móvil
• Aplicación Web
• Sistema externo

Todos consumiendo la misma API.

## Narración

"Antes de escribir una sola línea de código, quiero que entendamos qué es exactamente una Web API.

Imagina que tienes una aplicación móvil que necesita mostrar una lista de productos.

En lugar de conectarse directamente a la base de datos, envía una petición a una Web API.

La API recibe la solicitud, ejecuta la lógica necesaria y responde con la información correspondiente.

Gracias a este enfoque, diferentes aplicaciones pueden compartir la misma lógica de negocio de forma segura y organizada.

Durante toda esta academia construiremos precisamente esa capa de comunicación."

## Recursos

- Diagrama animado
- Iconos de computadora, navegador y teléfono
- Flechas animadas

## Takeaway

Una Web API es el puente de comunicación entre aplicaciones.

## Criterio de aprobación

El estudiante puede explicar con sus propias palabras qué es una Web API y para qué sirve.
