# Filosofía visual

Cada escena debe responder una única pregunta.

Cada escena debe tener un único objetivo visual.

Cada transición debe sentirse natural.

El estudiante nunca debe preguntarse:

"¿Qué está ocurriendo?"

La producción debe ayudar al aprendizaje, nunca distraerlo.
