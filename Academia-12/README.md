# Academia 12 - Inteligencia Artificial para Desarrolladores

Exploraremos cómo utilizar herramientas de Inteligencia Artificial para aumentar la productividad durante el desarrollo de software.

## Contenido

* ChatGPT
* GitHub Copilot
* Modelos de lenguaje
* Prompt Engineering
* Automatización
* Generación de código
* Revisión de código
* Pruebas automatizadas

## Proyecto Final

Asistente de desarrollo impulsado por IA.

## Objetivo

Aprender a integrar la IA como una herramienta de apoyo en el desarrollo profesional.
